# Superstore Sales Analysis

## Project Overview
This project analyzes the Superstore dataset to uncover valuable business insights through data cleaning, exploratory data analysis, visualization, and statistical hypothesis testing.

## Objectives
- Analyze sales performance and trends
- Identify top-performing products
- Evaluate category-wise sales
- Compare regional performance
- Understand customer segment contributions
- Validate findings through hypothesis testing

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analyses
- Sales Trend Analysis
- Top 10 Products by Sales
- Sales by Category
- Sales by Region
- Sales by Segment
- Hypothesis Testing

## Conclusion
The analysis highlights important business opportunities by identifying high-performing products, categories, regions, and customer segments. Statistical testing was used to support data-driven recommendations.
